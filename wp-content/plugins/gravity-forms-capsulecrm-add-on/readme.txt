== Description ==



> This plugin make the link beetween wordpress and capsuleCRM the web CRM via
the forms provided by Gravity. It requires <a href="http://capsulecrm.com/?referrer=DCCNKG" rel="nofollow">CapsuleCRM</a> 
and the premium version of the <a href="http://www.tastyburger.net/gravity" rel="nofollow">Gravity Forms plugin</a>.

###Integrate CapsuleCRM with Gravity Forms

####No more cutting and pasting data from one screen to another.
When you get form filled a lead will already be in your CapsuleCRM.

Simply install our free plugin, activate it, and you are moments away from taking
your lead forms to the next level.

####Gravity Forms CapsuleCRM Add-on will capture the following fields:

 - Name
 - Organization
 - Email
 - Address
 - Phone Number

Anything else will be added to a note field.

This plugin is free to use and configure you how you like. We ask that if you use
and enjoy this plugin then please either signup to <a href="http://www.tastyburger.net/capsule" rel="nofollow">CapsuleCRM</a>
or <a href="http://www.tastyburger.net/gravity" rel="nofollow">Gravity Forms plugin</a>
using our links or use the Donate button within the plugin.

####Features

 - Entities parametised in one file, could integrate opportunities, tasks, ...
 - Send in real time or planified via cronjob
 - Transformation and calculation capacities

== Installation ==
 
1. Upload plugin files to your plugins folder, or install using WordPress' built-in Add New Plugin installer
2. Activate the plugin
3. Go to the plugin settings page (under Forms > CapsuleCRM)
4. Enter the information requested by the plugin.
5. Click Save Settings.
6. If the settings are correct, it will say so.
7. Follow on-screen instructions for integrating with CapsuleCRM.
